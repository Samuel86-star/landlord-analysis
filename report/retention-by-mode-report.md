# 分玩法新增用户留存分析报告

> 分析周期：2026-02-10 ~ 2026-04-14
> 玩法映射：1=经典、2=不洗牌、3=癞子
> 口径：分子基于对局判定（区分"同玩法留存"与"整体留存"），分母为"新增且首日有对局"的用户
> 样本量（首日有对局）：经典 73,006 人 / 不洗牌 9,427 人 / 癞子 2,945 人

---

## 一、执行摘要

1. **不洗牌是留存最高的玩法，癞子反而最差**：不洗牌整体次留 25.92%、7 留 11.44%，领先经典（22.59% / 9.58%）；癞子整体次留 25.40% 与不洗牌齐平，但**同玩法次留仅 15.01%**，远低于不洗牌的 20.56%（-5.5pp）——癞子用户回来后大量跑去玩别的。
2. **癞子的"黏性缺口"最大**：同玩法/整体次留比值，经典 0.79、不洗牌 0.79、癞子仅 **0.59**，表明癞子作为"首日体验玩法"无法固化用户，次日倾向切换。
3. **首日多玩法用户 = 高留存**：只玩 1 种玩法次留 21.69%，玩 2 种跳到 27.75%（+6.1pp），玩 3 种达 33.23%（+11.5pp）；多玩法探索是留存最强的行为信号之一。
4. **最优倍数区间在三玩法间差异显著**：经典的最优区间是 12-48x（次留 20%），不洗牌"越高越好"——48x+ 次留 21.87% 是全区间最高；癞子同样 48x+ 最高（16.41%），但整体水平比不洗牌低 5pp，**高倍不等于高留存**。
5. **对局数 10 局+ 阈值普适**：三玩法的 10 局+ 次留均跃升至 29–33%，其中癞子 10 局+ 次留 33.11% 甚至高于经典，说明只要"玩得够多"，癞子留存能力不差；差距主要来自**一局就走**的流失（癞子 1 局占比 33.7%，远高于经典 11.7% 和不洗牌 16.6%）。
6. **经济影响在癞子玩法中最畸形**：经典/不洗牌的巨亏用户次留反而最高（经典 47.78%、不洗牌 28.77%，样本量小但趋势一致），而癞子巨亏用户次留仅 **1.64%**（n=244），这与癞子 360 渠道的数据污染高度相关（详见章节四）。
7. **首局即选癞子的用户次留最低**：首局选癞子的 100 人，同玩法次留仅 13.00%，低于首局选经典（17.81%）和不洗牌（17.35%）。不过首局选癞子的"误入其他玩法"的 30 人表现两极（n 过小）。
8. **连败阶梯规律在三玩法中完全一致**：连败越多次留越高，6-9 连败次留可达 33-38%，没有证据支持"差异化连败干预阈值"。

## 二、玩法留存基线对比

### 2.1 用户规模与首日参与

| 玩法 | 首日有对局用户数 | 占三玩法份额 | 首日平均对局数（经验估算） |
| ------ | ----------------- | ------------ | -------------------------- |
| 经典 play_mode=1 | 73,006 | 85.5% | 9-10 局 |
| 不洗牌 play_mode=2 | 9,427 | 11.0% | 7-8 局 |
| 癞子 play_mode=3 | 2,945 | 3.5% | 6-7 局（部分渠道仅 1-2 局） |

经典作为默认入口吸收绝大部分流量，不洗牌是第二梯队，癞子用户规模仅为经典的 4%——这直接决定了癞子分析的样本约束，很多渠道×分组的数据不可解读。

### 2.2 两个留存指标对比（同玩法 vs 整体）

| 指标 | 经典 | 不洗牌 | 癞子 |
| ------ | ------ | ------- | ------ |
| 同玩法次留 | 17.81% | **20.56%** | 15.01% |
| 整体次留 | 22.59% | **25.92%** | 25.40% |
| 同玩法 7 留 | 7.02% | **8.21%** | 4.69% |
| 整体 7 留 | 9.58% | **11.44%** | 10.22% |
| 同/整体次留比 | 0.79 | 0.79 | **0.59** |

**关键洞察**：癞子的整体留存（25.40%）其实与不洗牌基本持平，但用户"次日不再玩癞子"的比例高达 41%——远高于经典和不洗牌的 21%。**癞子扮演的是"新奇体验"而非"长期粘性"角色**。

### 2.3 跨渠道的一致性

选取 4 个主流渠道 看同玩法次留：

| 渠道 | 经典 | 不洗牌 | 癞子 | 差距(不洗牌-经典) |
| ------ | ------ | ------- | ------ | ------------------ |
| 华为（最大） | 20.29% | 22.44% | 18.07% | +2.15pp |
| vivo | 16.00% | 18.73% | 18.76% | +2.73pp |
| OPPO | 15.63% | 18.15% | 14.25% | +2.52pp |
| IOS | 19.50% | 15.75% | 16.67% | -3.75pp |
| 咪咕（离群） | 5.13% | 5.61% | 4.08% | +0.48pp |

- 大多数渠道中"不洗牌 > 经典 > 癞子"成立，荣耀渠道不洗牌异常突出（31.42%）。
- iOS 反例：经典 > 不洗牌，可能与 iOS 用户的玩法选择习惯有关。
- **咪咕是异常渠道**（全玩法次留仅 4-6%）、**360 渠道的癞子数据极异常**（次留 1.32%，见后续分析），数据质量需单独核查。

## 四、玩法内因子分析

### 分析发现卡片 1：倍数（M-04）

**【维度】** 首日平均倍数分组（≤6 / 6-12 / 12-24 / 24-48 / 48+）

**【发现】** 三玩法的同玩法次留随倍数变化：

| 倍数区间 | 经典 d1 | 不洗牌 d1 | 癞子 d1 |
| --------- | -------- | ---------- | --------- |
| ≤6 | 6.95% | 10.94% | 5.77% |
| 6-12 | 12.78% | 14.29% | 11.02% |
| **12-24** | **20.02%** | 16.79% | 8.01% |
| **24-48** | **20.09%** | 16.68% | 14.74% |
| **48+** | 14.75% | **21.87%** | **16.41%** |

**【玩法间差异】**

- 经典呈标准倒 U 型：12-48x 是"黄金区间"（d1 20%），48+ 回落到 14.75%。
- 不洗牌**单调递增**：越高倍次留越高，48+ 反而最优（21.87%）。这符合"不洗牌倍数天然偏高且受众适应高倍节奏"的预期。
- 癞子**也是 48+ 最高**（16.41%），但整体水平普遍比经典低 3-5pp，所谓的"癞子天然高倍"并没有转化为留存优势。
- 经典 vs 癞子在 12-24x 的差距最大（20.02% vs 8.01%，-12pp）——**癞子的中等倍数用户基本不回来**。

**【假设/解读】** 经典的倒 U 型符合"既要刺激又要可控"的新手心理，当倍数超过 48 单局波动过大反而驱赶新手。不洗牌和癞子用户自我选择性强，愿意进来的本身就偏好高倍体验。

**【建议】** 保持经典 12-48x 作为新手推荐倍数区间；**不必对不洗牌/癞子强行设倍数上限**，真实用户在高倍中反而留存更好。

### 分析发现卡片 2：胜率（M-05）

**【维度】** 首日胜率分组 (<30% / 30-40% / 40-50% / 50-60% / 60%+)

**【发现】** 三玩法均呈**倒 U 型**（40-50% 最优）：

| 胜率区间 | 经典 d1 | 不洗牌 d1 | 癞子 d1 |
| --------- | -------- | ---------- | --------- |
| <30% | 10.45% | 17.90% | 9.14% |
| 30-40% | 20.89% | 21.53% | 19.71% |
| **40-50%（最优）** | **25.65%** | **27.84%** | **26.71%** |
| 50-60% | 22.68% | 23.03% | 22.65% |
| 60%+ | 14.48% | 17.70% | 16.36% |

**【玩法间差异】** 最优峰值几乎一致（经典 25.65% / 不洗牌 27.84% / 癞子 26.71%），但两端差异显著：

- 低胜率（<30%）不洗牌依然有 17.90%，经典和癞子都只有 10% 左右——**不洗牌的容错能力最强**。
- 高胜率（60%+）三玩法都下降，说明"太容易赢"消磨新奇感在三玩法普适。

**【假设/解读】** 胜率对留存的影响**不存在玩法特异性**，40-50% 是心理甜点；不洗牌低胜率仍有留存可能因为"下一局可能翻盘"的预期延续性。

**【建议】** 新手匹配算法应把胜率目标统一锁定在 **40-55%**，不需按玩法差异化。

### 分析发现卡片 3：对局数（M-06）

**【维度】** 首日对局数（1 / 2-3 / 4-5 / 6-10 / 10+）

**【发现】** 三玩法同玩法次留：

| 对局区间 | 经典 d1 | 不洗牌 d1 | 癞子 d1 | 经典用户占比 | 癞子用户占比 |
| --------- | -------- | ---------- | --------- | ------------ | ------------ |
| 1 局 | 7.46% | 14.09% | 7.16% | 11.7% | **33.7%** |
| 2-3 局 | 9.81% | 14.96% | 12.59% | 16.0% | 23.7% |
| 4-5 局 | 11.48% | 20.30% | 13.56% | 15.4% | 12.8% |
| 6-10 局 | 16.84% | 20.33% | 19.25% | 25.5% | 14.5% |
| **10+ 局** | **29.66%** | **30.88%** | **33.11%** | 31.4% | 15.4% |

**【玩法间差异】**

- 10 局+ 时三玩法留存几乎追平（29-33%），**癞子在该区间甚至略高于经典**。
- 癞子最大的问题是**1 局即走的用户占 33.7%**，而经典仅 11.7%——癞子的"试一局就弃"是流失主因。
- 不洗牌在 4-5 局即达到 20%，拐点比经典（6-10 局）更早。

**【假设/解读】** 癞子规则复杂、首局挫败感强，导致大量"试玩一局退出"。但一旦用户越过 5 局门槛，癞子的娱乐性甚至高于经典。

**【建议】** 针对癞子新玩家设置**"玩够 5 局送奖励"任务**或新手引导，攻克其第 1 局流失痛点。

### 分析发现卡片 4：经济变化（M-07）

**【维度】** 首日净银子变化分组

**【发现】** 分玩法的同玩法次留对比：

| 经济变化 | 经典 d1 | 不洗牌 d1 | 癞子 d1 |
| --------- | -------- | ---------- | --------- |
| 巨亏 <-5 万 | 47.78%（n=90） | 28.77%（n=73） | **1.64%（n=244）** |
| 大亏 -5 万 ~ -1 万 | 21.98% | 22.74% | 10.23% |
| 小亏 -1 万 ~ 0 | 17.61% | 20.11% | 16.91% |
| 小赚 0 ~ 1 万 | 16.44% | 18.35% | 16.17% |
| 大赚 1-5 万 | 19.34% | 27.16% | 24.39% |
| 巨赚 >5 万 | 32.34%（n=807） | 36.63% | 27.27% |

**【玩法间差异】** 经典与不洗牌呈现"U 型"或"两端高"规律（巨亏和巨赚都刺激回流），**癞子的巨亏组次留仅 1.64% 是全报告最异常的数据点**：

- 该组 244 人中**仅 4 人次日回归**，且高度集中在 360 渠道（119 人）和 努比亚渠道（57 人）——与癞子 360 渠道整体异常（次留 1.32%）、努比亚渠道异常（2.25%）吻合。
- 若剥离 360 和 努比亚渠道，癞子巨亏组剩余样本量不足，无法得出一般性结论。

**【假设/解读】** 部分特定渠道的癞子用户可能是机器流量或刷量行为（极高倍数 1472x、仅 1-2 局即走、几乎零留存）。真实癞子用户的经济-留存关系应参照 华为、vivo 等主流渠道（次留 17-18%）。

**【建议】** 此因子需**按渠道过滤脏数据后**再做决策；主流渠道上"亏得多反而回来"的规律也适用于癞子，不必差异化。同时建议数据团队核查 360、努比亚渠道癞子数据源。

### 分析发现卡片 5：连败（M-08）

**【维度】** 首日最大连败长度（无连败 / 1-2 / 3-5 / 6-9 / 10+）

**【发现】** 同玩法次留：

| 连败长度 | 经典 d1 | 不洗牌 d1 | 癞子 d1 |
| --------- | -------- | ---------- | --------- |
| 无连败 | 10.43% | 11.54% | 10.98% |
| 1-2 连败 | 13.66% | 18.25% | 11.55% |
| **3-5 连败** | 25.55% | 25.58% | 22.98% |
| **6-9 连败** | **33.83%** | **38.13%** | **35.80%** |
| 10+ 连败 | 50.67%（n=75） | 50.00%（n=4） | 75.00%（n=4） |

**【玩法间差异】** 规律**高度一致**：三玩法都是"连败越多留存越高"，且 6-9 连败组的次留都在 33-38%；这与 M-06 的"对局数越多留存越高"本质上是同一现象（能经历多次连败说明玩得够久）。

**【假设/解读】** 连败不是留存的负向因子，反而是"深度参与"的代理变量。没有发现任何玩法需要差异化的连败干预阈值。

**【建议】** 不需要玩法差异化的连败保护策略，现有机制足够。

## 五、首局玩法选择 × 留存（M-09）

首次对局所在玩法（global_first_play_mode）与用户首日在各玩法的行为交叉：

| 首局玩法 | 同玩法用户 | 同玩法次留 | 整体次留 | 整体 7 留 |
| --------- | ---------- | ---------- | --------- | ---------- |
| 首局经典（n=72,724） | 是 | 17.81% | 22.56% | 9.57% |
| 首局经典但也玩不洗牌 | 否 | 22.31% | 29.20% | 13.19% |
| 首局经典但也玩癞子 | 否 | 15.15% | 25.57% | 10.23% |
| 首局不洗牌（n=3,320） | 是 | 17.35% | 19.91% | 8.28% |
| 首局不洗牌但也玩经典 | 否 | 18.22% | 29.74% | 11.52% |
| **首局癞子（n=100）** | 是 | **13.00%** | 21.00% | 4.00% |
| 首局癞子但也玩经典（n=11） | 否 | 36.36% | 45.45% | 18.18% |

**核心发现：**

- 绝大多数新手（首局用户中 95%+）默认进入经典，说明默认入口设计主导了首日选择。
- **首局选癞子的用户（100 人）是所有组中同玩法 7 留最低的（4.00%）**——这小群体要么是误操作，要么是抱着"试新鲜"心态、缺乏延续性的玩家。
- "首局选 A 玩法但也玩 B 玩法"的用户留存普遍高于"单玩 A"的用户（首局经典+玩不洗牌组次留 29.20% vs 纯经典 22.56%）——再次验证**多玩法探索是留存放大器**。

**建议**：对首局选择非经典玩法的新手，给予主动引导去尝试经典（默认玩法），促成多玩法探索。

## 六、三维交叉（M-11 倍数 × 胜率 × 玩法）

| 玩法 | 倍数 | 胜率 | 样本 | 次留 | 7 留 |
| ------ | ------ | ------ | ------ | ------ | ------ |
| **经典最优** | 12-24x | 40-50% | 2,957 | **27.90%** | 13.02% |
| 经典亚优 | 24-48x | 40-50% | 3,904 | 26.90% | 11.12% |
| 经典最差 | ≤12x | <40% | 2,467 | 7.74% | 3.04% |
| **不洗牌最优** | 12-24x | 40-50% | 30（样本小） | 36.67% | 26.67% |

**各玩法的"最佳画像"：**

- 经典：中倍（12-48x）+ 均衡胜率（40-50%）→ 次留 27%+

**差异化信号**：经典最优在中倍，不洗牌/癞子最优在高倍。三者的"胜率甜点"都在 40-50%。

## 七、综合结论与差异化建议

### 7.1 核心归因

1. **癞子的结构性问题**：不是留存绝对值低，而是"玩法忠诚度低" + "1 局即走比例高"。癞子是"玩过一次就够"的玩法，在新手定位上更接近"尝鲜"而非"主战场"。
2. **不洗牌是潜力股**：次留/7 留均为三玩法之首，但用户规模仅为经典的 13%。真正能从经典引导用户尝试不洗牌是留存提升的杠杆点。
3. **玩法组合 > 单玩法深度**：玩 3 种玩法用户次留比单玩法用户高出 11.5pp——多玩法探索的因果性（或强相关）比任何单因子更强。

### 7.2 是否需要差异化调整

| 参数 | 是否差异化 | 理由 |
| ------ | ---------- | ------ |
| 倍数推荐区间 | **需要** | 经典 12-48x / 不洗牌 48+ / 癞子 48+ |
| 胜率目标 | 不需要 | 三玩法均 40-50% 最优 |
| 连败保护阈值 | 不需要 | 连败本身为正向信号 |
| 对局数引导任务 | 需要（癞子特别） | 癞子 "玩够 5 局送奖励" 突破首局流失 |
| 新手默认玩法 | 保持经典 | 首局选癞子/不洗牌的用户留存反而更差 |

### 7.3 与全局分析的对比

全局分析中发现的 3 个核心规律：

1. **倒 U 型倍数**：在经典成立，**在不洗牌/癞子不成立**（变为单调递增）→ 需玩法拆分。
2. **40-50% 胜率最优**：在三玩法均成立 → 全局规律继续有效。
3. **10+ 局是留存拐点**：在三玩法一致（均在 29-33%）→ 全局规律继续有效。

全局分析最多只能看到"70% 用户集中在 24x+"的均值现象，拆到分玩法后才发现高倍是否利好取决于玩法本身——**分玩法拆分是必需的诊断动作**。

### 7.4 待解决问题

1. 癞子在 360、努比亚渠道的异常数据（次留 1-2%）需数据质量专项核查。
2. 首局选癞子的 100 人的画像需做前置行为分析（广告来源、点击路径等）。
3. M-10 玩法切换路径分析未执行，无法量化"经典→癞子"这类切换路径的留存效应。
